# Mood Tracker App

Mobile app to track daily mood and visualize weekly progress.

## Features
- Add / update daily mood (Sad, Normal, Happy)
- RecyclerView list of days and moods
- Progress chart built with MPAndroidChart
- MVVM + LiveData for reactive UI

## How to run
1. Open in Android Studio
2. Build and run on emulator or device (API 24+)
3. Press "Add Random Mood" or go to "Progress" to view chart
